# Video chat build with SimpleWebRTC and Vue.js

- 按钮关闭 变色

- 浏览器 不支持 则提示

- 画板 [fabric](https://github.com/fabricjs/fabric.js)

- 屏幕共享 screen sharing

- 回车发送消息

- 文件共享

- 富文本消息

- 对用户静音、关闭视频

- 收到消息提示，新加入用户提示

- 关闭视频 显示 已关闭

- 进网页若url没有room参数，则有两种选项：输入已有房间号、新建会议（生成会议号、二维码、分析）
## Project setup
```
npm install
```
and run

```
npm run serve
```
